# 🎬 AnimTube: AI Creative Studio

AnimTube — это современная веб-студии для создания персонажей и кадров с использованием мощных нейросетей **Google Gemini** и **xAI Grok**.

## ✨ Особенности
- **Двойной интеллект**: Переключение между Gemini и Grok одной кнопкой.
- **Библиотека персонажей**: Храните описания ваших героев локально.
- **Умный промпт**: Автоматическое объединение описания персонажа с описанием сцены.
- **Glassmorphism Design**: Стильный интерфейс с эффектами прозрачности и темной темой.

## 🚀 Как запустить как сайт (GitHub Pages)

Если вы загрузили этот репозиторий на GitHub, вы можете сделать его доступным по ссылке (сайт) за 30 секунд:

1. Откройте ваш репозиторий в браузере.
2. Перейдите во вкладку **Settings** ⚙️.
3. В левом меню выберите **Pages** 📄.
4. В разделе **Build and deployment** убедитесь, что выбрано "Deploy from a branch".
5. Выберите ветку `main` и папку `/ (root)`, нажмите **Save**.
6. Через 1-2 минуты ваша ссылка появится вверху этой страницы!

## 🔐 Безопасность
Все API-ключи хранятся исключительно в **LocalStorage** вашего браузера. Они никогда не отправляются на GitHub и не видны другим пользователям.
